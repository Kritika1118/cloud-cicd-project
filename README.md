# Automated CI/CD Pipeline for Cloud Application Deployment

## Technologies Used

- Docker
- Flask
- GitHub Actions
- Render
- Git
- Python

## Features

- Dockerized Flask application
- Automatic CI using GitHub Actions
- Automatic deployment to Render
- Containerized environment
- Version control with Git

## Workflow

Developer
      ↓
Git Push
      ↓
GitHub Actions
      ↓
Docker Build
      ↓
Render Deployment
